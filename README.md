#  Domino's Sales Performance Analysis — Power BI Dashboard

An interactive business intelligence dashboard built using Microsoft Power BI
to analyze Domino's pizza sales data, uncover revenue trends, and understand
customer ordering behavior.

---

##  Project Overview

This project analyzes a dataset of 48,620+ pizza sales records using Power BI Desktop.
The goal is to transform raw transactional data into actionable business insights
through interactive visualizations and KPI tracking.

---

##  Dashboard Pages

### Page 1 — Sales Performance Overview
- Total Revenue, Total Orders, Total Quantity Sold (KPI Cards)
- Revenue Distribution by Category (Pie Chart)
- Top 10 Pizzas by Revenue (Bar Chart)
- Revenue by Pizza Size (Column Chart)
- Monthly Sales Trend (Line Chart)
- Interactive Category Slicer (Chicken, Classic, Supreme, Veggie)

### Page 2 — Detailed Behavioral Analysis
- Total Orders by Hour of Day (Bar Chart)
- Total Orders by Day of Week (Bar Chart)
- Order Density by Day & Hour (Heatmap Table)
- Average Order Value (KPI Card)

---

##  Key Insights

-  Total Revenue: ₹817,860
-  Total Orders: 21,000+
-  Total Quantity Sold: 49,570+
-  Average Order Value: ₹38.31
-  Top Pizza: The Thai Chicken Pizza
-  Busiest Day: Friday (3,538 orders)
-  Peak Hours: 12:00–13:00 & 17:00–18:00
